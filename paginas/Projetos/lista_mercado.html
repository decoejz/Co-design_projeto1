Lista de Mercado

Curso: Desing de Software de

Período: 1° semestre

Data: 05/04/2016

Duração: 2 semanas

Descrição: Neste projeto, os alunos deveriam fazer um programa na linguagem de programação python, que servisse para o usuário como a montagem de uma lista de mercado para ele, onde pudessem ser adicionado elementos a lista e seus preços, retirar elementos da lista, ver o total da compra e alterar o valor de algum item que ja esteja na lista. Como estavamos no começo do curso, o código era básico porém com uma dificuldade que faria com que os alunos tivessem que ir em busca de conhecimentos externos para conseguir finalizar o projeto.

Conhecimentos exigidos: Aprender a aprender, programação na linguagem python, lógica.
